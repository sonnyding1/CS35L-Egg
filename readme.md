# CS35L Egg

This project is UCLA's CS35L final project.

## Installation

Follow these steps to install and run this project:

1. **Clone the repository**

```bash
git clone git@github.com:sonnyding1/CS35L-Egg.git
```

2. **Navigate to the directory**

```bash
cd CS35L-Egg
```

3. **Install the dependencies**

For the backend:

```bash
cd backend
npm install
```

For the frontend:

```bash
cd frontend
npm install
```

4. **Run the project**

For the backend:

```bash
cd backend
npm run dev
```

For the frontend:

```bash
cd frontend
npm run dev
```
